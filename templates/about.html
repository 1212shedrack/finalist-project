{% trans "Fighting Amaranthus Diseases with AI" %}

Amaranthus is a highly nutritious crop cultivated widely across Africa and Asia. However, it is susceptible to several diseases that reduce yield and quality. Early detection is critical to preventing crop loss and protecting farmer livelihoods.

This system uses a state-of-the-art deep learning model trained on over 3,136 leaf images to classify Amaranthus leaf diseases with 98%+ accuracy. Farmers can simply take a photo and receive instant, expert-level recommendations.

{% trans "Diseases We Detect" %}

{% trans "Our model classifies Amaranthus leaves into four categories" %}

{% trans "Healthy" %}

Normal healthy Amaranthus leaf with no signs of disease or infection.

  • Uniform green color
  • No spots or lesions
  • Smooth leaf surface
{% trans "Low Risk" %}
{% trans "Leaf Spot" %}

Fungal disease causing dark necrotic spots on leaf surfaces, reducing photosynthesis.

  • Dark brown/black spots
  • Yellowing around spots
  • Leaf wilting
{% trans "Medium Risk" %}
{% trans "White Rust" %}

Oomycete pathogen (Albugo bliti) causing white powdery pustules. Spreads rapidly.

  • White chalky patches
  • Distorted leaves
  • Premature leaf drop
{% trans "High Risk" %}
{% trans "Non-Amaranthus" %}

Image does not contain an Amaranthus leaf the model cannot provide reliable results.

  • Wrong plant species
  • Unclear/blurry photo
  • Non-leaf objects
{% trans "Unknown" %}

EfficientNet B0

Compact yet powerful CNN architecture. Achieves 98%+ accuracy with only 5.3M parameters — perfect for fast inference.

TensorFlow Lite

Optimized model format for production deployment. Runs inference in milliseconds without GPU requirements.

Django Backend

Robust Python web framework handling image uploads, model inference, database storage, and recommendations.

This system was developed as an academic research project aimed at supporting smallholder Amaranthus farmers through artificial intelligence. The model was trained on a carefully curated dataset of 3,136 annotated leaf images using transfer learning with EfficientNet B0.
